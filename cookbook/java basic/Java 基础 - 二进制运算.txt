
#运算符号
> & 与运算
a & b , a和b都是1的情况结果为1 (1&1=1), 其他情况为0

运用
将不需要位数设置为0
比如 11110101 只需要后四位
11110101 & 1111


> 或运算 
a | b,  a和b其中一个为1则为1,其他情况为0

---------------------------------------
与众不同的异或运算
---------------------------------------
因为在二进制中同一个位置只有0,1, 所以天生拥有了明确的对立面特性，于是也产生了对应的异或运算

0 ^ 0 = 0, 两位相同
1 ^ 1 = 0, 两位相同
1 ^ 0 = 1, 两位不相同
0 ^ 1 = 1, 两位不相同
